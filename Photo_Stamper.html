What is EXIF Stamper?
EXIF Stamper burns metadata — GPS coordinates, reverse-geocoded address, date, and time — directly onto your photos as visible text. Everything runs locally in your browser. No photos are ever uploaded to any server.
Layout
Left panelLoaded images list — click to select, hover for controls CenterLive canvas preview — drag or click to reposition the stamp Bottom barStamp settings — fields, font, alignment, color, position presets Right panelEXIF data table, geocoded address, and manual override fields
✓ Tip

The stamp is non-destructive during editing — the original file bytes are preserved. The stamp is only baked in when you download.

Loading Photos
Drag & Drop
Drag photos or a .zip file directly onto the app. You can also drag from the file list to reorder or move to groups.
Click to Browse
Click the drop zone in the bottom-left to open a file picker. Accepts JPG, PNG, WEBP, HEIC, TIFF, BMP, GIF and .zip files.
ZIP Files
When a ZIP is loaded, photos are automatically grouped by subfolder. Photos in the ZIP root are placed ungrouped.
File List Controls
  • Click any photo to select and preview it
  • Checkbox — uncheck to download without stamp
  • ⇄ button (hover) — move photo to a different group
  • ⠿ handle (hover) — drag to reorder or move to group
  • ✕ button (hover) — remove photo from list

Click anywhere on the canvas to jump the stamp there. Drag the stamp ghost to fine-tune position.

Geocoding
GPS coordinates are automatically converted to street addresses when photos load.
Default: Census Bureau + Nominatim
No API key required. For US addresses, the US Census Bureau geocoder is queried first — it uses TIGER/USPS data and accurately resolves house numbers. For non-US addresses (or when Census finds no match), it falls back to Nominatim (OpenStreetMap).
Optional Paid Providers
LocationIQ5,000/day free — set GEOCODER='locationiq' and LOCATIONIQ_KEY in the file OpenCage2,500/day free — set GEOCODER='opencage' and OPENCAGE_KEY in the file
⚠ Note

Geocoding requires an internet connection. Photos without GPS show "No GPS data". The Census geocoder only covers US addresses.
