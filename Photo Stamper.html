What is EXIF Stamper?
EXIF Stamper burns metadata — GPS coordinates, reverse-geocoded address, date, and time — directly onto your photos as visible text. Everything runs locally in your browser. No photos are ever uploaded to any server.
How it works
  • Load photos (or a ZIP file) into the app
  • The app reads each photo's EXIF metadata automatically
  • GPS coordinates are reverse-geocoded to a street address
  • A text stamp is drawn onto the canvas preview
  • Download individual photos or all photos as a ZIP
Layout
Left panelLoaded images list — click to select, hover for controls CenterLive canvas preview — drag or click to reposition the stamp Bottom barStamp settings — fields, font, alignment, color, position presets Right panelEXIF data table, geocoded address, and manual override fields
✓ Tip

The stamp is non-destructive during editing — the original file bytes are preserved. The stamp is only baked in when you download.

Loading Photos
Drag & Drop
Drag photos or a .zip file directly onto the drop zone in the bottom-left corner. You can drop multiple files at once.
Click to Browse
Click the drop zone to open a file picker. Accepts JPG, PNG, WEBP, HEIC, TIFF, BMP, GIF and .zip files.
ZIP Files
When a ZIP is loaded, photos are automatically grouped by subfolder. Photos in the ZIP root are placed ungrouped. The ZIP filename becomes the group name for root-level photos.
File List
  • Click any photo to select and preview it
  • Checkbox on each photo — uncheck to download without stamp
  • ⇄ button (hover) — move photo to a different group or ungroup it
  • ✕ button (hover) — remove photo from the list
✓ Tip

Photos are processed in parallel. Geocoding runs immediately in the background as photos load. The "Download all" button enables once all addresses are resolved.

Date & TimeShows date and time from EXIF (format: MM/DD/YYYY hh:mm:ss AM/PM) GPSDecimal latitude and longitude from EXIF AddressReverse-geocoded street address from GPS coordinates FilenameOriginal filename of the photo
Typography
SizeFont size in pixels (12–120px). Scales with the actual image size on download. FontChoose from 8 typefaces. Each option previews in its own font. B / I / UBold, italic, underline toggles AlignLeft, center, right, or justify — aligns text within the stamp block
Color
Text colorClick the white swatch to open the color picker. Choose from palette or enter a custom hex value. BG colorBackground fill behind the stamp text BG opacitySlider controls background transparency (0% = fully transparent)
✓ Tip

Default is white text with a dark semi-transparent background (50% opacity). This works well on both light and dark photos.

Stamp Positioning
Drag to Position
Click anywhere on the canvas preview to jump the stamp to that point. Then drag the stamp ghost (dashed outline) to fine-tune its position.
Preset Grid
The 3×3 grid in the Position column snaps the stamp to 9 common positions: corners, edges, and center.
Per-Photo Independence
Each photo remembers its own stamp position independently. Moving the stamp on one photo does not affect others.
Apply Position to Group
The ↓ pos to group button (below the preset grid) copies the current photo's stamp position to every other photo in the same group. This is a one-time copy — photos remain independent afterwards.
✓ Tip

Position is stored as a ratio (0–1) of the image dimensions, so it stays proportionally correct regardless of image size or orientation.

Groups
What are groups?
Groups are collections of photos — typically from the same job site, folder, or ZIP subfolder. They appear as collapsible headers in the file list with a colored dot.
Creating Groups
  • From ZIP — subfolders become groups automatically
  • Manually — hover a photo, click ⇄, then choose "New group…"
Group Controls
⬇ on headerDownload all photos in that group as a ZIP (GroupName.zip with photos inside GroupName/ folder) ✕ on headerDissolve the group — photos move to ungrouped ▾ chevronCollapse or expand the group in the list
Moving Photos
Hover any photo and click the ⇄ button to move it to another group, create a new group, or ungroup it. Ungrouped photos appear at the top of the list with no header.
Manual Override — Entire Group
In the Manual Override panel, switch scope to Entire group then click ↓ Apply to all in group to apply the current photo's overrides to all photos in the group.
Manual Override
The Manual Override panel (right sidebar) lets you correct or replace any metadata field for individual photos or entire groups.
Fields
DateOverride the EXIF date. Format: MM/DD/YYYY TimeOverride the EXIF time. Format: HH:MM:SS GPSOverride GPS coordinates. Accepts: 26.184133,-80.263028 or 26.184133 -80.263028 AddressOverride the geocoded address with any text. Will NOT change GPS coordinates.
Auto-fill behavior
  • Type GPS → Address field auto-fills from reverse geocoding
  • Clear GPS → Address restores from EXIF GPS coordinates
  • Clear Address → Address restores from GPS (manual or EXIF)
  • Type Address → GPS field is NOT changed (address is independent)
Scope
This photoOverride applies only to the currently selected photo Entire groupShows when photo is in a group with 2+ photos. Click "↓ Apply to all in group" to propagate.
⚠ Note

Manual overrides are saved per-photo and persist when you switch between photos. They are also written into the downloaded JPEG's EXIF metadata.

Downloading
Download This Image
The ⬇ This image button downloads the currently previewed photo with the stamp burned in and original EXIF metadata preserved.
Download All Stamped
The ⬇ Download all stamped button packages every loaded photo into a single stamped_photos.zip. Photos in groups are placed in subfolders matching the group name. Ungrouped photos go in the ZIP root.
Download Group
Click ⬇ on a group header to download only that group's photos as GroupName.zip, with all photos inside a GroupName/ folder.
Stamp Checkbox
Each photo has a checkbox in the file list. Unchecked photos are downloaded without any stamp — the original file bytes are used as-is.
Geocoding Gate
The "Download all" button shows ⏳ Geocoding X/Y… while addresses are being resolved. It enables automatically when all photos have been geocoded.
✓ Tip

EXIF metadata is fully preserved in downloads. Manual date, time, and GPS overrides are also written back into the saved JPEG's EXIF tags.

Geocoding
Geocoding converts GPS coordinates into a human-readable street address. It runs automatically in the background when photos are loaded.
Default Provider
Nominatim (OpenStreetMap) is used by default — completely free, no API key required, no usage limits. Requests are spaced 1.1 seconds apart to respect Nominatim's usage policy.
Alternative Providers
LocationIQ5,000 requests/day free. More accurate for some regions. Requires free API key from locationiq.com. Falls back to Nominatim at limit. OpenCage2,500 requests/day free. Requires API key from opencagedata.com.
Switching Providers
Open the HTML file in a text editor and change the config at the top of the script:
var GEOCODER = 'locationiq';
var LOCATIONIQ_KEY = 'pk.your_key_here';
Geocoding Cache
Addresses are cached by coordinate. If multiple photos share the same GPS location (rounded to 5 decimal places), only one geocoding request is made.
⚠ Note

Geocoding requires an internet connection. Photos without GPS coordinates will show "No GPS data" and are skipped in the geocoding queue.
